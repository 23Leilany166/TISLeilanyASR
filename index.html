Sistema de Gestión de la Constructora. (SGC)

El Sistema de Gestión de la Constructora (SGC) es un sistema propuesto para optimizar la administración en el sector de la construcción, diseñado para gestionar de manera eficiente las áreas del negocio: Clientes, Trabajadores, Planos, Proveedores.

Problemática del proyecto

En MASON_CONSTRUCTORA, se identificó la necesidad de un sistema centralizado para gestionar los recursos de los proyectos. Antes de implementar la base de datos, la información sobre trabajadores, maquinaria y materiales estaba dispersa, lo que causaba confusión y retrasos.

Objetivo del proyecto

Desarrollar un sistema de información centralizado para optimizar los procesos operativos, mejorar la eficiencia en la gestión de recursos y facilitar la toma de decisiones en la empresa MASON_CONSTRUCTORA, utilizando herramientas tecnológicas para el diseño de la interfaz de usuario y administración de la base de datos

Diagrama

Subsistemas del proyecto

El sistema se organiza en subsistemas utilizando la estrategia de "divide y vencerás", con el objetivo de identificar y gestionar eficientemente cada componente del proyecto de construcción. Esto permite que las distintas unidades operen de manera autónoma, garantizando su integración en un conjunto cohesivo y funcional.
