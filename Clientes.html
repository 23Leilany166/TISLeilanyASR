Clientes

El subsistema de Clientes permite registrar, consultar y actualizar la información de los clientes que contratan los servicios de la constructora. Su objetivo principal es mantener actualizada la información de los clientes y facilitar un seguimiento eficiente de los proyectos.

Campos de Clientes

Los principales campos que se registran para cada cliente son los siguientes:

ID Cliente Nombre Apellido Paterno Apellido Materno Correo Teléfono Dirección Sexo Fecha de nacimiento Edad Eliminar
Código único que identifica a cada cliente dentro del sistema. Nombre completo del cliente para su registro e identificación. Correo electrónico del cliente, usado para comunicación y notificaciones. Número de contacto telefónico para comunicación directa. Dirección física del cliente para ubicar proyectos, entregas o correspondencia. Sexo del cliente (masculino/femenino) para fines estadísticos y registros. Fecha de nacimiento del cliente para cálculos de edad y segmentación. Edad calculada automáticamente a partir de la fecha de nacimiento. Opción que permite eliminar del sistema a un cliente cuando ya no sea necesario conservar sus datos.

Permisos por rol

Supervisor:
  • El supervisor debe de ser capaz de:
    • Registrar información de nuevos clientes:
        ID Cliente Nombre Apellido Paterno Apellido Materno Correo Teléfono Dirección Sexo Fecha de nacimiento Edad Eliminar
    • Consultar información de clientes:
    • Buscar y visualizar los datos registrados utilizando campos como:
      • ID_Cliente
      • Nombre
    • Actualizar información existente:
    • Actualizar la informacion utilizando los siguients campos:
      • ID_Cliente
      • Nombre
    • Eliminar la informacion de un cliente:
    • Eliminar la informacion utilizando los siguientes campos:
      • ID_Cliente
      • Nombre
    Ingeniero Civil: El Ingeniero Civil debe de ser capaz de:
  • Consultar la información utilizando:
    • ID_Cliente
    • >Nombre
