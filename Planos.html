Planos

El subsistema de Planos permite registrar, consultar y actualizar toda la información relacionada con los planos de construcción utilizados en los proyectos. El objetivo es gestionar y mantener actualizados los planos de construcción para evitar errores en obra, reducir riesgos asociados a información desactualizada y asegurar que los responsables cuenten con los documentos correctos en el momento adecuado.

Campos de Planos

Los principales campos que se registran para cada plano son los siguientes:

ID_Planos Código Calle Teléfono Número Proyecto asociado Encargado Bloquear Código Eliminar Código
Código único que identifica a cada plano dentro de la base de datos. Código de referencia del plano. Calle donde se llevará a cabo el proyecto. Número de contacto del encargado del proyecto. Número de identificación del plano. Nombre del proyecto asociado al plano. Nombre del encargado del plano. Opción que permite bloquear el código del plano para futuras modificaciones. Opción que permite eliminar el plano del sistema cuando ya no sea necesario.

Permisos por rol

Permisos por rol

Arquitecto:
  • El arquitecto debe de ser capaz de:
    • Asignar un plano a un proyecto
    • Consultar planos utlizando: Codigo, ID Planos
    • Registar avance de un plano utlizando: Codigo, ID Planos
    Supervisor:
  • El supervisor debe de ser capaz de:
    • Revisar y aprobar un plano para que sea asignado a un proyecto
    • Registrar planos utilizando: ID Planos, Codigo
    • Consultar planos utlizando: Codigo, ID Planos
    • Actualizar información de planos ulizando: ID_Planos
    • Eliminar planos utilizando: ID_Planos
