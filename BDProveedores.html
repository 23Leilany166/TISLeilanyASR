Descripción de la Base de Datos de Proveedores

Propósito General

La base de datos de Proveedores está diseñada para gestionar la información de los proveedores de la constructora. Permite registrar, consultar, actualizar y bloquear proveedores según las necesidades del proyecto.

Tabla Proveedores

Propósito

Contiene la información clave de cada proveedor. Facilita la gestión de relaciones comerciales y el control de estado activo/bloqueado.

Campos

  • ID_Proveedor: Clave primaria. Se autogenera (IDENTITY).
  • Nombre: Obligatorio.
  • Apellido_Paterno: Apellido paterno del cliente.
  • Apellido_Materno: Apellido materno del cliente.
  • Correo: Obligatorio y único.
  • Teléfono: Obligatorio.
  • Dirección: Obligatoria.
  • Sexo: Obligatorio.
  • Fecha_Nacimiento: Obligatoria.
  • Bloqueado: Indica si el proveedor está activo (0) o bloqueado (1).

Código SQL para crear la tabla


CREATE DATABASE ProveedoresBASE;
USE ProveedoresBASE;

-- Tabla de Proveedores
CREATE TABLE PROVEEDORES (
    ID_Proveedor INT PRIMARY KEY IDENTITY(1,1),
    Nombre NVARCHAR(100) NOT NULL,
    Apellido_Materno NVARCHAR(50);
    Apellido_Materno NVARCHAR (50);
    Correo NVARCHAR(100) NOT NULL,
    Telefono NVARCHAR(15) NOT NULL,
    Direccion NVARCHAR(255) NOT NULL,
    Sexo NVARCHAR(10) NOT NULL,
    Fecha_Nacimiento DATE NOT NULL,
    Bloqueado BIT NOT NULL DEFAULT 0
);
